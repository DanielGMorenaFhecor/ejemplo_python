Example repository for the FIB Python Library project

Instructions:
    1. First of all, open this current project as the current Workspace.
    2. If working with Conda, no package installations are needed. Go to 4th step.
    3. If working with a global environment or in a virtual environment, then run te following command to install the packages: 'pip install -r requirements.txt'
    4. Open the notebook_example.ipynb to check a real example of how to use these libraries
